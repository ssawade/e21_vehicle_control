MIL Simulation

1) Start parameter script (parameters.m) in folder ".\MatlabSimulink\".
2) Open main model (main_STM32_H743_MIL.slx) in folder ".\MatlabSimulink\MIL\"
3) Set configuration of referenced model "ECU_RCP" in Model Explorer from "Configuration_HIL" to "Configuration_MIL" for MIL simulation.
